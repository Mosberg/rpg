## Overview

This mod overhauls Minecraft villages and villagers with RPG and simulation features. It is modular, data-driven, and highly configurable.

### Key Features

- Dynamic village generation and new village types
- Smarter villagers with memory, needs, and relationships
- Professions, quests, reputation, and economy systems
- Defense, raids, and world simulation (diplomacy, caravans)
- In-game config via Mod Menu

## Configuration

All major systems are configurable via Mod Menu or JSON files in `assets/rpg/`, `data/rpg/` and `config/`.

### Main Config Options

- Enable/disable subsystems (AI, professions, events)
- Difficulty (survival realism, raid frequency)
- Performance (AI tick cycles, population cap)
- Customization (growth speed, trade balance, personality variety, hardcore mode)

## Extensibility

- JSON files define professions, trades, quests, and structures
- Other mods can patch into events, trade tables, and AI goals
